Restaurant Analysis using SQL
This is a mini project that explores a dataset of restaurants,Sales and Delivery , using SQL.
 The dataset contains information about various restaurants, including their location, cuisine, ratings, and other relevant data.

Getting Started
To run this project, you'll need to have an SQL database management system installed on your local machine. You can use any of the popular database management systems such as MySQL,Workbench, PostgreSQL, or SQLite.

Once you have an SQL database management system installed, you can create a new database and import the provided dataset file into it. 
The dataset file is available in the data directory in the project files.

Data
The dataset used in this project is available in the data directory. It contains Multiple CSv file based on Restaurants that contains information about various restaurants.

Usage
To run the project, simply execute the SQL queries provided in the queries.sql file in the project files. The SQL queries provide various analysis and insights on the dataset.

You can execute these SQL queries on your local machine using any SQL client or interface of your choice, such as MySQL Workbench, pgAdmin, or SQLiteStudio.
